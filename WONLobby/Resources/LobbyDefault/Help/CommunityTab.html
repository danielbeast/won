Community Information - The Community Tab

The Community Tab displays news, images, and links to keep you up to date on the latest aspects of the game and it's multiplayer. Visit this tab to get the latest info on updates, new content, and fan sites.

The Community Tab functions much like your favorite Internet Browser. The current page is displayed and you may click on links to view additional pages. Below the current page are typical browser buttons:

  • Back Button. Returns you to the previous page. Back is only available after clicking a link.
  • Forward Button. Returns you to next page. Forward is only available after using Back.
  • Stop Button. Stops the load of the current page. Stop is only available while a page is loading.
  • Refresh Button. Reloads the current page.
  • Home Button. Returns you to the community home page.

Additionally, you may search through the currently displayed page using the Find controls, located in the lower right. To search the current page, enter a string and select Find Next to search forward or Find Prev to search backward. Searches are case-insensitive, unless the Case Sensitive check box is checked.
