Direct Tab

The Direct Tab is used to join and host direct connection (connect by IP address) games. It contains a Game Browser that lists any direct connect games you have defined. In addition, there are two additional buttons available:

  • Add. Displays the Add dialog to add a new direct connection game to the game browser. The dialog allows you to specify the game address as IP:port and an optional name. If no name is entered, the address will also be used as the name. When all information is entered, click Add to add the new game to the browser. Or you may click Cancel to abort.

  • Remove. Remove is only available when a game is selected in the game browser. Click Remove to delete the selected game from the game browser permanently. Note that you may also right-click on a game and select Remove to delete it as well.

Note: you must have an active Internet connection in order to play direct connection games.
