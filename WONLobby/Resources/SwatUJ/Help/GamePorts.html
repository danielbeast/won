Game Ports for Online Play

The following ports must be open in order play online:

Incoming/Outgoing Ports

Port Protocol Purpose
**Matchmaking Port** TCP, UDP Pre-game matchmaking
**Game Port(s)** TCP Game play

Outgoing Ports

Port Protocol Purpose
80 HTTP Web requests
15101 TCP Master server communication
15200 TCP Authentication
15300 TCP Machine connection detection
15800 TCP Account maintenance
**Lobby Port(s)** TCP Lobbies

Outgoing ports are subject to change; it is recommended that all outgoing communication be allowed when you play online.
